# Product and UX planning

These files are design references, not runtime web assets. The runnable frontend only serves files from `src/frontend/public`.

## Contents

- `ui-ux/`: screen mockups, UI overview, color-theme board, and detailed theme table.
- `architecture/`: system architecture, API mapping, ERD, data-flow diagrams, logic flow, and user journey.
- `brand/`: original logo explorations and light/dark variants.
- `media/`: cinematic product-video references.

## Implementation notes

- The four planned themes are Light, Dark, Night, and Synthwave.
- The canonical color tokens and component rules are summarized in the root `DESIGN.md`.
- The theme-planning PDF checklist is a planning artifact; implementation status should be verified against `src/frontend/src/index.css` and the theme components.
- Runtime-ready logos and hero artwork are maintained separately in `src/frontend/public/assets`.
